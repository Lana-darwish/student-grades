# Student Grades System 🎓

A Python program to manage and evaluate student grades.

## Features
- Store student information (name, age, grade)
- Display student details
- Check if student is passing or failing

## How to run
```bash
python student.py
```

## Example output
my name is lana, my age is 22, my grade is 98
pass ✅
my name is sara, my age is 20, my grade is 78
pass ✅

## Built with
- Python 3
- OOP (Object Oriented Programming)
